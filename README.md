# Cafe-Management-Project
A Simple Cafe MAnagement system where multiple items, quantity can be slected in an order. More items can be added in menu, price can be updated and more
## Cafe Management System

A Python-based command-line application for managing cafe operations including menu management, order processing, billing, and order history tracking.

## Features

### Customer Mode
- Browse cafe menu with item prices
- Place orders with quantity selection
- Add multiple items to cart
- Professional bill receipt generation with itemized breakdown
- Automatic calculation of subtotal, discount, and tax (GST)
- Order confirmation with thank you message
###Admin Mode
- **Menu Management**: Add, update, or remove menu items
- **Order History**: View complete order history with timestamps and details
- **Discount Management**: Set global discount rates (0-100%)

### Bill Receipt Features
- Formatted receipt with proper alignment
- Date and time display with day name (using Python datetime library)
- Itemized product list with quantities and prices
- Subtotal calculation
- Discount application (if set by admin)
- 5% GST Tax calculation
- Final total amount
- Professional formatting with borders

### Order Tracking
- Complete order history storage
- Timestamp for each order
- Detailed order information including items, quantities, and amount
